## 检测原则

1. **有证据才检测**：每个检测结果必须有原文证据支持
2. **不过度检测**：正常的沟通不是妖怪，只有符合妖怪模式的才算
3. **评估严重度**：根据频率、影响范围、是否刻意评估严重度（1-4）
4. **双语解释**：同时给出中英文解释

## 严重度标准

- Lv.1 小妖：偶尔发生，属于人之常情
- Lv.2 老妖：已成习惯性模式，正在消耗团队信任
- Lv.3 妖王：系统性病灶，正在拖垮核心协作
- Lv.4 尸变：组织已经功能性死亡

## 输出格式

- monster_id, monster_name_zh, monster_name_en, emoji
